# ELO - Comunidade & Conexão

Rede social focada em conexões reais e nostalgia técnica, inspirada na estética clássica das redes de 2010.

## 🚀 Tecnologias
- HTML5 / CSS3 (Design Black & Gold)
- JavaScript (Integração de autenticação)
- Supabase (Backend e Banco de Dados)

## 🔒 Segurança e Privacidade
O ELO foi construído sob o lema "Você é o dono dos seus dados", com foco total em privacidade e segurança técnica.
